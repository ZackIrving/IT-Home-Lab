# IT Home Lab (CompTIA A+ Focus)

## Overview
This repository documents my hands-on IT home lab as I prepare for the CompTIA A+ certification. The lab focuses on building foundational IT skills such as operating systems, networking, and troubleshooting, with plans to expand into cybersecurity.

## Goals
- Build strong IT fundamentals (A+ level)
- Practice real-world troubleshooting scenarios
- Gain hands-on experience with Windows and Linux
- Document all work to demonstrate skills to employers
- Transition into cybersecurity after foundational knowledge

## Lab Environment
- Hyper-V / VirtualBox
- Windows VM (planned / in progress)
- Ubuntu Server VM
- Basic networking (NAT, IP configuration)

## Lab Structure

### Phase 1 – Operating Systems & Setup
- VM setup (Windows + Linux)
- User management
- File systems and permissions

### Phase 2 – Networking Fundamentals
- IP addressing, DNS, DHCP
- Network troubleshooting
- Basic connectivity testing

### Phase 3 – IT Support & Troubleshooting
- Simulated real-world issues
- Fixing common IT problems
- System performance troubleshooting

### Phase 4 – Intro to Security (Later Phase)
- Linux hardening
- Basic monitoring tools
- Log analysis

## Notes
This lab is part of my transition into IT, with the goal of landing a Help Desk role and progressing into cybersecurity.
