# cybersecurity-home-lab

## Overview
This repository documents my personal cybersecurity home lab, designed to practice hands-on skills in areas such as Linux administration, network security, auditing, log analysis, and SOC workflows. The lab is built incrementally, from foundational setup to advanced detection and SIEM integration.

## Lab Objectives
- Learn and practice Linux server administration for security purposes
- Configure monitoring and auditing systems (auditd, rsyslog, UFW)
- Simulate attacks and practice detection workflows
- Set up a SOC-style environment with SIEM tools (Wazuh/ELK)
- Document all steps to demonstrate practical skills and methodology

## Lab Structure
The repository is organized into phases:
- `phase-1-foundation.md` – Initial VM setup, SSH access, core tools, and baseline security
- `phase-2-networking.md` – Network configuration, firewall rules, and internal segmentation
- `phase-3-logging-siem.md` – Centralized logging, SIEM integration, and monitoring
- `phase-4-attack-simulations.md` – Attack simulation, log analysis, and detection exercises
- `phase-5-advanced.md` – Optional cloud integration and cross-platform SOC scenarios

## Notes
This lab is for personal learning purposes only.
